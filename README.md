# AutoPlan-DX

**AutoPlan-DX**는 확산 아키텍처(diffusion architecture)를 기반으로 한  
**공정 스케줄링 최적화** 팀 프로젝트 **AutoPlan-D**를 개인적으로 확장·발전시킨 프로젝트입니다.

본 프로젝트는 기존 팀 프로젝트의 핵심 아이디어를 바탕으로,
보다 **모듈화되고 확장 가능한 구조**로 재설계하고
확산 기반 최적화 기법에 대한 탐구를 심화하는 것을 목표로 합니다.

---

## 프로젝트 개요
현대 제조 및 공정 계획 환경에서는 복잡한 제약 조건과
동적인 변화에 유연하게 대응할 수 있는 스케줄링 전략이 요구됩니다.
AutoPlan-DX는 이러한 문제를 해결하기 위해
**확산 모델 기반 아키텍처를 활용한 공정 스케줄링 최적화 방법**을 탐구합니다.

본 레포지토리는 다음과 같은 개인 개발 목적을 가지고 있습니다.

- 기존 팀 프로젝트 구조의 재정비 및 리팩토링
- 공정 스케줄링 파이프라인의 확장성과 실험 가능성 향상
- 확산 기반 최적화 기법에 대한 구조적·실험적 연구

---

## 프로젝트 배경
